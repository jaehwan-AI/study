1. 이상치 정리 / 제어, 처리, 수정

-> 표준점수로 변환 후 -3 이하 및 +3 이상 제거
-> IQR(사분위수) 방식 : 1분위수 - 1.5*IQR 이하 및 3분위수 + 1.5*IQR 이상 제거
-> 수치형 데이터의 경우 평균(mean)이나 중앙치(median), 0으로 대체
-> 범주형 데이터의 경우 최빈값(mode)으로 대체
-> 예측 모델을 통한 예측값(predict)으로 대체

2. 결측치 정리 / 제어, 처리, 수정

-> 결측치 제거
-> 수치형 데이터의 경우 평균(mean)이나 중앙치(median), 0으로 대체
-> 범주형 데이터의 경우 최빈값(mode)으로 대체
-> 예측 모델을 통한 예측값(predict)으로 대체

3. 전처리 하는 목적 / 종류

-> 목적 : 일반적으로 수집된 데이터는 결측치나 이상치(noise)가 섞여있으며, 결측치나 이상치는 모델의 학습을 방해하고 모델의 성능을 저하시킨다. 따라서 전처리를 통해 모델의 성능을 저하시키는 요소를 처리해주어야 한다.

-> 종류
1) 데이터 정제 : 없는 데이터는 채우고 잡음은 제거, 모순된 데이터는 맞는 데이터로 교체하는 작업
2) 데이터 통합 : 여러 데이터 파일을 통합하는 작업
3) 데이터 축소 : 샘플링을 통해 데이터의 양을 줄이거나 분석 대상의 차원을 줄이는 작업
4) 데이터 변환 : 데이터를 정규화 또는 집단화하는 작업, z-score, log, mean을 구해서 사용하는 작업
5) 데이터 이산화 : 데이터를 구간으로 변환하는 작업
